Real Time Bus Tracker



The objective of this of this assignment was to create a button and display the live location of the MBTA bus route betweeen the campuses of MIT and Harvard. Once the button ["Show Stops Between MIT and Harvard"] is clicked, the marker on the map pinpoints each bus stop along the route, displaying real-time data of the bus. This was accomplished by using a token key (received from https://www.mapbox.com) and replacing "YOUR_KEY" in the "MapAnimation.js" file.
